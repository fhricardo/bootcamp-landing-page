Web Dev Bootcamp 2024

Sua melhor e maior oportunidade do ano para aprender sobre desenvolvimento web com os maiores nomes do mercado.

O que você vai aprender:

  • Domine a Base da Web com HTML5

    Com HTML5, você aprenderá a estrutura essencial de qualquer site ou aplicação web. Entenda como organizar conteúdos com tags semânticas, criar links, inserir imagens, e muito mais. Descubra como usar recursos modernos, como áudio, vídeo e formulários interativos. HTML5 é a fundação para criar experiências web acessíveis e eficazes.

  • Estilize o Mundo Digital com CSS3

    Com CSS3, transforme páginas simples em designs impressionantes. Aprenda a manipular cores, fontes e layouts, usando técnicas como flexbox e grid. Descubra como criar animações e transições que encantam os usuários. Domine estilos responsivos para adaptar sites a diferentes dispositivos, garantindo uma experiência visual perfeita.

  • Torne a Web Interativa com JavaScript

    Aprenda JavaScript para adicionar interatividade às suas páginas. Descubra como manipular elementos DOM, capturar eventos e criar funções dinâmicas. Desenvolva aplicações que respondem ao usuário em tempo real, conectando-se a APIs externas e explorando frameworks como React ou Vue.

  • Controle Seus Projetos com Git

    Com Git, você terá total controle sobre as versões do seu código. Aprenda a criar repositórios, rastrear mudanças e trabalhar em equipe de forma eficiente. Domine comandos para commits, branches e merges, e garanta segurança e organização no desenvolvimento dos seus projetos.

  • Colabore e Exponha no GitHub

    Descubra o poder do GitHub para armazenar, compartilhar e colaborar em projetos. Aprenda a criar repositórios públicos e privados, utilizar issues para gerenciar tarefas e revisar pull requests. Ganhe visibilidade ao publicar seus trabalhos e explore projetos de outros desenvolvedores para aprender ainda mais.
